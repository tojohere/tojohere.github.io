Inference runs via Core ML on the Apple Neural Engine (iOS) or ONNX Runtime CPU on Android (avg 46.5ms on Snapdragon 8 Elite). LiteRT with Hexagon NPU support is in place for future Qualcomm AI Hub compilation. Every step on-device — no internet required.

The ML Pipeline

Die ML-Pipeline

MLパイプライン

ML 파이프라인

Step 1

Translate

Google's GoEmotions dataset (58K Reddit texts, 28 labels) translated to DE/JA/KO using NLLB-200 on Apple Silicon.

Step 2

Augment

Merge native-language datasets: GermEval (10K DE), WRIME (35K JA), KOTE (50K KO) for authentic emotional context.

Step 3

Fine-Tune

XLM-RoBERTa-base fine-tuned per language on RunPod GPU (RTX 4090). 10 epochs, PBKDF2 learning rate warmup.

Step 4

Quantize & Ship

ONNX + INT8 dynamic quantization (1.1GB FP32 → 266MB INT8) for ONNX Runtime on Android, plus Core ML conversion via coremltools for the Apple Neural Engine on iOS.
